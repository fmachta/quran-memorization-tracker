🔐 دخول المشرف

أدخل رمز GitHub للوصول إلى لوحة إدخال البيانات. يتم تخزين الرمز في متصفحك فقط ولا يغادر جهازك.

كيف تحصل على الرمز؟
1. اذهب إلى github.com/settings/tokens
2. اختر Generate new token (classic)
3. امنح صلاحية repo فقط
4. انسخ الرمز والصقه هنا

⚠️ هام: الرمز يخزن في متصفحك فقط. لا تشاركه مع أحد.
